# 用户管理页面设计

## 1. 页面布局
采用标准 Dashboard 布局：`Sidebar` + `Header` + `Content Area`。

### 1.1 列表页 (`/users`)
- **顶部操作栏**：
  - 左侧：标题 "用户管理" + 副标题 "系统用户及权限配置"。
  - 右侧：
    - 搜索框 (Input + Icon): Placeholder="搜索姓名或邮箱..."
    - "新增用户" 按钮 (Button): Icon=<Plus>
- **数据表格 (DataTable)**：
  - 列定义：
    - `用户信息`：组合显示 Avatar + Name + Email (主列)
    - `角色`：Badge (Blue for ADMIN, Gray for USER)
    - `注册时间`：Format "YYYY-MM-DD"
    - `操作`：Dropdown Menu (编辑, 删除)
- **底部**：
  - 分页组件 (Pagination)

### 1.2 编辑/新增表单 (Sheet)
使用右侧抽屉 (`Sheet`) 组件，保持上下文。

- **Header**：标题 "新增用户" / "编辑用户"。
- **Body**：
  - `姓名` Input
  - `邮箱` Input (Type=email)
  - `密码` Input (Type=password, 编辑模式下 Placeholder="留空保持不变")
  - `角色` Select (USER, ADMIN)
- **Footer**：
  - "取消" Button (Outline)
  - "保存" Button (Primary, Loading state)

## 2. 交互细节
- **Loading**：
  - 表格加载时显示 Skeleton。
  - 表单提交时按钮变灰并显示 Spinner。
- **Feedback**：
  - 成功：Toast "用户已创建" / "更新成功"。
  - 失败：Toast "操作失败: 邮箱已被注册"。
- **Dialog**：
  - 删除操作触发 `AlertDialog`。

## 3. 路由与文件结构
- 页面: `app/users/page.tsx`
- 客户端组件: `app/users/user-client.tsx` (列表与状态)
- 表单组件: `app/users/user-sheet.tsx`
- Server Action: `app/actions/users.ts`
