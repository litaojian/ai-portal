# 项目管理功能需求文档

## 1. 功能概述
实现一个完整的项目管理模块，允许用户创建、查看、更新和删除（CRUD）项目信息。该模块将无缝集成到现有的 Dashboard 中。

## 2. 数据模型 (Project)
核心实体 `Project` 将包含以下字段：

| 字段名 | 类型 | 必填 | 默认值 | 说明 |
| :--- | :--- | :--- | :--- | :--- |
| **id** | String (CUID) | 是 | - | 主键 |
| **name** | String | 是 | - | 项目名称 |
| **description** | String | 否 | - | 项目详细描述 |
| **leader** | String | 否 | - | **(新增)** 项目负责人 |
| **budget** | Float/Decimal | 否 | 0 | **(新增)** 项目预算金额 |
| **status** | String | 是 | "进行中" | 状态 (进行中, 已完成, 暂停, 未开始) |
| **priority** | String | 是 | "中" | 优先级 (高, 中, 低) |
| **startDate** | DateTime | 是 | Now | 开始日期 |
| **endDate** | DateTime | 否 | - | 结束日期 |
| **createdAt** | DateTime | 是 | Now | 创建时间 |
| **updatedAt** | DateTime | 是 | Now | 更新时间 |

## 3. 用户流程 (User Stories)
1.  **查看列表**：用户点击侧边栏“项目管理”，进入列表页，查看包含分页、搜索和筛选（状态/优先级）的项目表格。
2.  **新建项目**：点击右上角“新建项目”按钮，从右侧滑出抽屉 (Sheet)，填写信息并保存。
3.  **编辑项目**：点击列表行或操作菜单中的“编辑”，滑出抽屉并回显当前信息，修改后保存。
4.  **删除项目**：点击“删除”按钮，弹出确认对话框，确认后删除数据。

## 4. 界面与交互设计
*   **交互模式**：**Sheet (侧边抽屉)** 用于所有表单交互（新建/编辑），保持上下文不丢失。
*   **列表设计**：
    *   复用 `components/data-table.tsx` 的风格。
    *   增加“预算”列的货币格式化显示。
    *   增加“状态”和“优先级”的视觉区分（不同颜色的 Badge）。

## 5. 技术实现路径
*   **Database**: 更新 Prisma Schema，迁移数据库。
*   **Backend**: 使用 Server Actions (`app/actions/project.ts`) 处理 CRUD 逻辑。
*   **Frontend**:
    *   页面: `app/dashboard/projects/page.tsx`
    *   表单组件: `components/project-form-sheet.tsx`
    *   验证: Zod schema
