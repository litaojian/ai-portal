# 项目管理模块 - 页面设计文档

## 1. 页面概览
*   **页面名称**：项目管理 (Projects)
*   **路由路径**：`/projects`
*   **功能目标**：提供项目信息的增删改查 (CRUD) 操作入口。

## 2. 页面布局与交互

### 2.1 列表页 (List View)
*   **布局结构**：
    *   **Header**: 面包屑导航 `Dashboard > 项目管理`。
    *   **Toolbar**: 
        *   左侧：搜索框 (Search Input)，支持按项目名称搜索。
        *   右侧：`+ 新建项目` 按钮 (Primary Button)。
    *   **Data Table**: 占据主要区域，展示项目列表。
    *   **Pagination**: 底部居右，翻页控制器。

*   **列表字段定义**：
    | 列标题 | 数据源 | 显示样式 | 交互 |
    | :--- | :--- | :--- | :--- |
    | **项目名称** | `name` | 粗体文本 | - |
    | **负责人** | `leader` | 头像(Avatar) + 姓名 | - |
    | **状态** | `status` | Badge (颜色区分：进行中-黑/蓝, 已完成-灰, 暂停-红) | - |
    | **优先级** | `priority` | 彩色文本 (高-红, 中-黄, 低-绿) | - |
    | **预算** | `budget` | 货币格式 (¥1,000.00) | 右对齐 |
    | **开始日期** | `startDate` | YYYY-MM-DD | - |
    | **操作** | - | `...` 下拉菜单 | 编辑 (打开 Sheet), 删除 (弹窗确认) |

### 2.2 新建/编辑抽屉 (Sheet)
*   **触发方式**：
    *   **新建**：点击 Toolbar 的“新建项目”按钮。
    *   **编辑**：点击列表行操作菜单中的“编辑”。
*   **容器**：Shadcn `Sheet` 组件，从屏幕右侧滑出，宽度 `sm:max-w-[540px]`。
*   **表单布局**：
    1.  **基本信息区**：
        *   项目名称 (Input, 必填)
        *   负责人 (Input, 选填)
    2.  **属性配置区**：
        *   状态 (Select: 进行中/已完成/暂停/未开始)
        *   优先级 (Select: 高/中/低)
    3.  **资源与时间区**：
        *   预算 (Input type=number, 前缀 ¥)
        *   开始日期 (DatePicker) | 结束日期 (DatePicker)
    4.  **详细信息区**：
        *   描述 (Textarea, 多行文本)
    5.  **底部操作栏**：
        *   `保存` 按钮 (提交表单，带 Loading 状态)

### 2.3 删除确认 (Alert Dialog)
*   **触发方式**：点击操作菜单中的“删除”。
*   **内容**：
    *   标题：“确认删除？”
    *   描述：“此操作无法撤销。该项目及其所有数据将被永久删除。”
    *   按钮：`取消` (Secondary), `删除` (Destructive/Red)。

## 3. UI 组件库参考
*   **Framework**: Next.js 16 + Tailwind CSS
*   **Components**: Shadcn/UI
    *   `Table`, `Button`, `Input`, `Select`, `Sheet`, `Form`, `Calendar`, `Popover`, `AlertDialog`, `Avatar`, `Badge`, `DropdownMenu`
*   **Icons**: Lucide React (`Plus`, `MoreHorizontal`, `Pencil`, `Trash2`, `CalendarIcon`)
