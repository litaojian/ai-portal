# 应用管理页面设计

## 1. 路由结构
- **列表页**: `/apps`
- **操作**: 使用侧边抽屉 (Sheet) 进行新增和编辑，保持上下文。

## 2. 页面布局 (List Page)

### 2.1 顶部区域
- **标题**: "应用管理"
- **面包屑**: Home > Apps
- **操作栏**: 
    - 搜索框 (Search Input)
    - 状态筛选 (Select: 全部/已上架/下架)
    - "新增应用" 按钮 (Primary Button) -> 触发 Sheet

### 2.2 数据区域
- **组件**: `DataTable` (Shadcn/UI)
- **列定义**:
    - **图标/名称**: 组合显示，图标在左，名称加粗，下方显示简短描述。
    - **状态**: Badge (Green: Published, Grey: Offline, Yellow: Draft)。
    - **版本**: 文本。
    - **开发者**: 文本。
    - **更新时间**: 格式化日期。
    - **操作**: 
        - 编辑 (Icon: Pencil，默认显示：显示) -> 触发应用编辑表单
        - 上架/下架 (Dropdown Menu)
        - 集成密钥 (Icon: Lock，默认显示：显示) -> 触发申请集成密钥表单
        - 删除 (Icon: Trash, Red) -> 弹窗确认是否继续

## 3. 交互设计 (Sheet Form)

### 3.1 表单内容
- **基本信息**:
    - 名称 (Input, Required)
    - 链接 (Input, URL)
    - 类型 (Select, 内部应用或集成应用)
    - 图标 (Select/Input, 暂支持输入 Lucide 图标名或 URL)
    - 描述 (Textarea)
- **管理信息**:
    - 状态 (Select: Draft, Published, Offline)
    - 版本 (Input)
    - 开发者 (Input)
- **OIDC客户端信息**:
    - 应用ID（Input）
    - 应用密钥 (Input)
    - 回调URL (Input，URL)
    - 授权模式 (Input, Checkbox，可选项：参考grantTypeOptions)    
### 3.2 验证 (Zod)
- 名称: min(2)
- URL: url() (optional)
