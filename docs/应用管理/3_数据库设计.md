# 应用管理数据库设计

## 1. 模型定义 (Prisma Schema)

```prisma
model Application {
  id          String   @id @default(cuid())
  name        String   // 应用名称
  description String?  // 应用描述
  icon        String?  // 图标 (URL 或 Icon Name)
  url         String?  // 访问链接 (e.g., /apps/chatgpt 或 外部链接)
  
  status      String   @default("draft") // draft (开发中), published (已上架), offline (已下架)
  version     String?  @default("1.0.0") // 版本号
  developer   String?  // 开发者/负责人
  category    String?  // 分类 (e.g., 文本, 图像, 助手) - 可选，预留

  createdAt   DateTime @default(now())
  updatedAt   DateTime @updatedAt
}
```

## 2. 字段说明

| 字段名 | 类型 | 必填 | 说明 |
| :--- | :--- | :--- | :--- |
| id | String | 是 | CUID 主键 |
| name | String | 是 | 应用显示的名称 |
| description | String | 否 | 简短介绍 |
| icon | String | 否 | 用于 UI 展示的图标 |
| url | String | 否 | 点击后跳转的地址 |
| status | String | 是 | 状态控制显隐 |
| version | String | 否 | 版本标识 |
| developer | String | 否 | 责任人 |
