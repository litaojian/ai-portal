# 应用管理功能需求

## 1. 背景
为了丰富 AI 门户的内容，需要一个功能来管理集成的各种 AI 应用。管理员可以注册新应用、管理其状态（上架/下架），并维护应用的基础信息。

## 2. 功能列表

### 2.1 应用列表
- 展示所有已登记的应用。
- 支持按状态（已上架、下架、开发中）筛选。
- 支持按应用名称搜索。
- 分页展示数据。

### 2.2 新增应用
- 填写应用基础信息：名称、描述、图标、访问链接、版本号、开发者、OIDC客户端相关信息。
- 初始状态默认为“开发中”或“已上架”。


### 2.3 编辑应用
- 修改除 ID 外的所有信息。
- 快捷操作：上架（Published）、下架（Offline）。


### 2.4 删除应用
- 删除应用记录（需二次确认）。

## 3. 业务规则
- **状态流转**：
    - Draft (开发中) -> Published (已上架)
    - Published (已上架) -> Offline (已下架)
    - Offline (已下架) -> Published (已上架)
- **必填项**：应用名称、访问链接。
- **唯一性**：应用名称建议唯一（非强制，但推荐）。

## 4. 权限
- 只有管理员或具有“应用管理”权限的角色可以操作（v1版本暂不限制细粒度权限，依赖登录态）。
